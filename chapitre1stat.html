Outils de la physique statistique

Introduction

La physique statistique vise à établir un lien entre les propriétés microscopiques des particules et les phénomènes macroscopiques observables. Elle s'applique à des systèmes composés d'un nombre très élevé de particules, comparable au nombre d'Avogadro, et utilise les principes de la mécanique quantique pour décrire leur comportement.

  • Macro-physique :

Mécanique classique ou relativiste : Étude des mouvements des corps.

Électromagnétisme : Décrit par les équations de Maxwell.

Thermodynamique : Basée sur les trois principes fondamentaux (énergie, entropie, température).

  • Microphysique :

Mécanique quantique : étude de comportement des particules à des dimensions où les lois classiques ne s’appliquent plus, notamment à l’échelle atomique et subatomique.

Rappels sur l’analyse combinatoire

La factorielle

La factorielle d'un entier n, notée n!n! , est le produit de tous les entiers positifs inférieurs ou égaux à n.

n!=n(n−1)(n−2)…..1=∏i=1nin! = n(n - 1)(n - 2)\ldots..1 = \prod_{i = 1}^{n}i

Exemple : 4!=4×3×2×1=24

Permutations

Une permutation est un arrangement de tous les n objets distincts, en tenant compte de l'ordre. Le nombre de permutations est donné par :Ann=n!\ A_{n}^{n} = n!

Exemple : Pour 3 objets M, N, P,, il y a 3!=6 permutations possibles (MNP, MPN, NMP, NPM, PMN, PNM).

Arrangements

Un arrangement de p objets parmi n (avec p ≤n ) est le nombre de manière qu’on peut ordonner p objets différents choisis parmi n.
Autrement dit, c’est la manière de choisir et d’ordonner p objets parmi n, en tenant compte de l’ordre.

Ce le nombre, noté AnpA_{n}^{p} est défini par :

Anp=n!(n−p)!A_{n}^{p} = \frac{n!}{(n - p)!}\ =n(n−1)(n−2)…..(n−p+1)\ n(n - 1)(n - 2)\ldots..(n - p + 1)

Combinaison de p parmi n

Une combinaison de p objets parmi n ne tient pas compte de l'ordre. Le nombre de combinaisons est

Cnp=n!(n−p)!p!=Anpp!C_{n}^{p} = \frac{n!}{(n - p)!p!} = \frac{A_{n}^{p}}{p!}

Exemple : Pour 4 objets et p=2, il y a 4!(4−2)!2!\frac{4!}{(4 - 2)!2!}​ =6 combinaisons possibles.

\mathbf{\ }Elément de la théorie de probabilité

La probabilité P(A) d'un événement A est le rapport entre le nombre de cas favorables et le nombre total de cas possibles (supposés équiprobables).

A toute évènement A on associe P(A)P(A)\ : la probabilité pour que cette évènement se réalise.

On distingue deux cas :

  • Ensemble d’évènements discrets

Les évènements discrets sont ceux qu’on peut distinguer les uns des autres sans ambiguïtés. Ces évènements sont indépendants. Si on réalise N expériences donnant chacun un résultat et si nin_{i} est le nombre de fois où l’évènement i se réalise parmi N. on définit la probabilité de cet évènement par : Pi=lim⁡N→∞niNP_{i} = \lim_{N \rightarrow \infty}\frac{n_{i}}{N}

  • Ensemble d’évènements continu : densité de probabilité

Les évènements continus sont déterminés avec une certaine incertitude. Soit Δp(x)\mathrm{\Delta}p(x) la probabilité pour que l’évènement (mesure d’une longueur l) soit dans l’intervalle [x,x+Δx]\lbrack x,x + \mathrm{\Delta}x\rbrack. Soit Δn(x)\mathrm{\Delta}n(x) le nombre de fois parmi N mesures où l appartient [x,x+Δx]\lbrack x,x + \mathrm{\Delta}x\rbrack on a :

Δp(x)=limN→∞Δn(x)N\mathrm{\Delta}p(x) = \lim_{N \rightarrow \infty}\frac{\mathrm{\Delta}n(x)\ }{N}

La probabilité pour que la longueur l est dans l’intervalle [x,x+Δx]\lbrack x,x + \mathrm{\Delta}x\rbrack est :

dp(x)=ρ(x)dxdp(x) = \rho(x)dx
